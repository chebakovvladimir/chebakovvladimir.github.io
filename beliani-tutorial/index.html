Для создания нового роут-компонента - создать папку в 'название внешнего роута'/components/pages/'название роута'. В этой папке создать файл index.js и 'название роута'View

Правило поведения при создании компонента
  • Использовать camelCase при создании переменных, функций, названий файлов
  • В одном файле - один компонент
  • Название файла должно совпадать с названием компонента, кроме случая, когда файл называется index.js
  • Все перемнные связанные с this объявляются в начале файла
  • Далее - Конструктор, если нужен
  • Далее - метод componentDidMount, если нужен
  • Далее - все остальные методы lifecycle реакта
  • Далее - все остальные методы
  • Далее - метод render
  • В конце файла прописывается propTypes
  • Метод рендер должен занимать не больше 50 строк
  • state используется для временного кеширования данных или для того чтобы отображать всплывающие элементы, во всех остальных случаях использовать props
